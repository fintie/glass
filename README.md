# Qwen Glass AI - Premium Landing Page

A modern, responsive, and beautifully designed landing page for Qwen Glass AI - an enterprise platform combining voice, vision, and local LLM models for Education, Healthcare, and Software Development.

## 🎨 Design Features

- **Modern Premium Design** inspired by Apple Vision Pro, Humane AI Pin, and Google Glass
- **Glassmorphism Effects** with modern gradients and frosted glass aesthetics
- **Fully Responsive** - Mobile, Tablet, and Desktop optimized
- **Smooth Animations** - Scroll effects, hover interactions, and page transitions
- **Professional Startup Style** - Suitable for investors, enterprise customers, and technical partners
- **High Performance** - Optimized for fast loading and smooth scrolling

## 🚀 Getting Started

### Option 1: Open with Live Server (VS Code)

1. Install the "Live Server" extension in VS Code
2. Right-click on `index.html` and select "Open with Live Server"
3. The page will open in your default browser

### Option 2: Open Directly

Simply open `index.html` in any modern web browser.

### Option 3: Deploy to Web

Upload all files to your web hosting provider:
- GitHub Pages
- Netlify
- Vercel
- AWS S3
- Any standard web hosting

### Animations
- **Fade In Effects** - Elements fade in on page load and scroll

## 🛠️ Customization

### Change Brand Colors
Edit `:root` variables in `styles.css`:
```css
:root {
    --primary-color: #0066ff;
    --secondary-color: #ff6b35;
    --accent-color: #00d4ff;
    /* ... more colors */
}
```

### Update Company Information
